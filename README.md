# FlashcardMiniProgram
基于微信小程序实现的自定义单词基于程序（和 Anki 一样）

使用技术  
1.使用 Taro3 和 Vue3 开发、Vuex 状态管理  
2.使用 echarts 可视化历史复习数据  
3.采用微信云开发作为后端，云存储用户数据  
4.运用 SM-2 间隔记忆算法动态规划词牌的复习时间
